# 1. 项目介绍
**适用于前后端分离的后端快速开发平台**

**基于Spring Boot 2.0版本，集成MyBatis、Spring Security、Redis、Thymeleaf、Swagger、Quartz、ActiveMQ、
Log4j2高性能日志框架、邮件发送、多线程接口等功能的快速开发平台，下载即可开发。**
